1 Column Row Example

One column example. Scroll down the page to see a variety of columns layouts, maybe you can find the one you were looking for.

The content is centered because i added a class called .column-center to my responsive-extensions.css. You can remove that class to fall back to defaults. Every H1 tag has a line gif on sides. To remove it delete the class .line.. In this page every row has a container div in a way you can give a background color or pattern to all the row size and also to apply a row shadow with the class .row-shadow. Each column is made with the new Bootstrap 3 class .col-lg-* where the asterix is the column width calcutated on a 12 columns grid system.
